To implement these five experiments, a programmer needs to focus on three phases: Stimulus Construction, the Familiarization Phase (Learning), and the Test Phase (Evaluation).Core Architecture & StimuliShapes: Use 24 distinct, novel geometric shapes.Color Assignment: For each participant, randomly assign 12 shapes to a "Red" pool and 12 to a "Green" pool.Triplet Structure: Within each color, divide the 12 shapes into 4 "triplets" (fixed 3-shape sequences, e.g., $ABC, DEF, GHI, JKL$).Interleaving: Generate two independent streams (Red and Green). Interleave them randomly into one long sequence of 624 shapes.Constraint: The number of remaining shapes in one color cannot exceed the other by more than 6.Timing: Standard Stimulus Onset Asynchrony (SOA) is 1,000ms (800ms shape + 200ms blank ISI), unless specified otherwise.Experiment 1A: The Baseline Attention TestThis experiment tests if selective attention is required for Visual Statistical Learning (VSL) using a fast presentation rate.Timing: 400ms SOA (200ms shape + 200ms blank ISI).Familiarization (Cover Task): Instruct the user to watch shapes of only one color (e.g., "Press spacebar if a Red shape repeats").Note: Repetitions occur within the color stream but may be separated by interleaved shapes of the other color.Test Phase (2IFC): Present two 3-shape sequences in black.Sequence 1: A valid triplet from the familiarization phase.Sequence 2: A "foil" (3 shapes from that color that never appeared in that order).User Action: Choose which sequence feels more "familiar".Experiment 1B: Increased ExposureThis experiment tests if a slower pace (more processing time) allows learning of unattended shapes.Modification: Change the SOA from 400ms to 1,000ms.Procedure: Identical to Exp 1A in every other way, including the black-color test shapes.Experiment 2A: Maintaining Color ContextThis experiment tests if keeping the original colors during the test phase helps retrieval of the unattended patterns.Modification: In the Test Phase, shapes are no longer black.If a shape was Green during learning, it must be Green during the test.Timing: 1,000ms SOA.Experiment 2B: Swapping Color ContextThis experiment tests if learning is "abstract" or tied specifically to the color features.Modification: In the Test Phase, reverse the colors.Shapes that were Green during learning are now shown in Red, and vice versa.Timing: 1,000ms SOA.Experiment 3: Implicit Response Time (RT) MeasureThis experiment replaces the "familiarity" question with a speed-based task to see if VSL improves performance even without conscious awareness.Familiarization Phase: Same as Exp 1B (1,000ms SOA, color-based cover task).Test Phase (Detection Task): * Show a "target" shape (e.g., Shape B).Play a rapid stream of 24 black shapes (200ms shape + 200ms blank).The user presses a key as fast as possible when they see the target.Logic for Programmer: Track the reaction time (RT) based on the target's position in its original triplet.If learning occurred, the user should be faster at detecting the 2nd and 3rd shapes of a triplet compared to the 1st, because the 1st shape "primes" the others.
